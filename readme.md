# Mining Plutonium

All the plutonium are belong to us. Pluto is a planet.

## Installation

### With Composer

```
$ composer require kingfnips/Plutonium
```

```json
{
    "require": {
        "kingfnips/Plutonium": "0.0.*"
    }
}
```

```php
<?php
require_once __DIR__ . '/vendor/autoload.php';

use kingfnips\Plutonium\Plutonium;

$miner = new Plutonium;
```
